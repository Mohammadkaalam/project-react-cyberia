# 📝 To-Do List App

یک وب‌اپلیکیشن مدیریت لیست وظایف ساخته شده با **React, Redux Toolkit, TypeScript, Bootstrap, React Router v6, Vite**  
این پروژه امکان دسته‌بندی، فیلتر، مرتب‌سازی، حالت شب، انیمیشن، و ذخیره‌سازی داده‌ها در مرورگر را فراهم می‌کند.

---

## 🚀 ویژگی‌ها

- 📌 **مدیریت وظایف**: افزودن، ویرایش، حذف، علامت‌گذاری به عنوان انجام‌شده/انجام‌نشده
- 📂 **دسته‌بندی وظایف**: شخصی، کاری، همه
- 🔍 **فیلتر بر اساس دسته‌بندی**
- ↕ **مرتب‌سازی وظایف**: جدیدترین، قدیمی‌ترین، بر اساس دسته‌بندی
- 🌙 **حالت شب (Dark Mode)**
- 🔔 **اعلان موفقیت/خطا** (Toast Notifications)
- 🎯 **نمایش تعداد کل وظایف و انجام‌شده**
- 🗑 **پاک کردن همه وظایف انجام‌شده**
- 🎬 **انیمیشن افزودن/حذف وظایف**
- 💾 **ذخیره‌سازی در localStorage**
- 📱 **ریسپانسیو برای موبایل و دسکتاپ**

---

## 🛠 تکنولوژی‌ها

- [React](https://reactjs.org/)
- [Redux Toolkit](https://redux-toolkit.js.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Bootstrap](https://getbootstrap.com/)
- [React Router v6](https://reactrouter.com/)
- [Vite](https://vitejs.dev/)

💡 راهنمای استفاده
با دکمه افزودن وظیفه یک تسک جدید اضافه کنید.

از منوی دسته‌بندی برای فیلتر وظایف استفاده کنید.

از بخش مرتب‌سازی ترتیب نمایش را تغییر دهید.

برای فعال کردن حالت شب، روی آیکون 🌙 کلیک کنید.

برای حذف همه وظایف انجام‌شده، از دکمه پاک کردن انجام‌شده‌ها استفاده کنید.
